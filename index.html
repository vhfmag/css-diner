Você está prestes a aprender sobre seletores CSS! Seletores são como você define em que elementos aplicar estilos.

Amostra 1 - Uma regra CSS

p {
   margin-bottom: 12px;
}

Aqui, "p" é o seletor (seleciona todos os elementos <p>) e aplica o estilo margin-bottom.

Para jogar, digite um seletor CSS no editor abaixo para selecionar os itens corretos na mesa. Se você acertar, você vai avançar para o próximo nível.

Passe o mouse sobre os itens na mesa para ver o HTML deles.
